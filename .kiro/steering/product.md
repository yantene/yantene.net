# Product Overview

Yantene is a modern full-stack web application platform built on React Router 7 with server-side rendering, designed for deployment on Cloudflare Workers edge infrastructure.

## Core Capabilities

- **Server-Side Rendering (SSR)**: Delivers fully rendered pages with bot detection and streaming responses for optimal SEO and performance
- **API Integration**: Seamless coexistence of API endpoints and frontend routes through Hono backend middleware
- **Edge Deployment**: Production-ready deployment on Cloudflare Workers global network with built-in observability
- **Type-Safe Development**: End-to-end TypeScript with strict mode across frontend, backend, and workers
- **AI-DLC Workflow**: Kiro-style spec-driven development with structured steering and specification management

## Target Use Cases

- Full-stack applications requiring SSR and API endpoints on the same domain
- Edge-first web applications leveraging Cloudflare's global network
- Type-safe React applications with strict code quality standards
- Projects requiring structured AI-assisted development workflows

## Value Proposition

Combines React Router's modern SSR capabilities with Cloudflare Workers' edge performance, while maintaining strict type safety and code quality standards through comprehensive ESLint configuration. The AI-DLC workflow enables structured, specification-driven development with persistent project memory.
